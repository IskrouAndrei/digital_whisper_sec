Название проекта: "CyberSentry: The Digital Whisper"
Твой личный ИИ-агент для доминирования в информационном поле кибербезопасности.

Миссия
Предоставлять профессиональному сообществу (SOC-аналитикам, DevOps-инженерам, пентестерам) критически важную информацию о киберугрозах на 2-3 часа раньше, чем это сделают классические новостные ресурсы. Минимизировать время на рутинный мониторинг и перевод, фокусируя внимание эксперта только на актуальных угрозах.

Ценностное предложение (УТП)
Скорость (Speed-to-Feed): Автоматизированный конвейер, где новость от публикации в зарубежном источнике до поста в канале проходит путь менее чем за 10 минут.

Техническая точность: LLM-агент, настроенный на сохранение терминологии (CVE, RCE, APT), не допускающий "опопсения" технических деталей, что критически важно для ИБ-специалистов.

Омниканальность: Единый центр управления контентом. Сформировал и одобрил пост — он автоматически разлетелся по Telegram, VK, X, LinkedIn и Threads.

Аналитическая глубина: Интеллектуальный дайджестинг. Вместо ежедневного потока шума — еженедельная структурированная аналитика с хардкорным разбором уязвимостей для Хабра.

Ключевые возможности (MVP)
Умный парсинг: Мониторинг ключевых мировых центров кибербезопасности (BleepingComputer, The Hacker News, Unit 42, Securelist и др.).

ИИ-рерайт и перевод: Адаптация контента под специфику платформ (от короткого саммари для X до полноценных дайджестов для LinkedIn).

Контроль качества: Система "Human-in-the-loop" — ни один пост не уйдет в паблик без подтверждения администратора.

Сверхзвуковое оповещение: Система алертинга: если сервис падает или API возвращает ошибку, бот незамедлительно оповещает администратора.

Виральный фильтр: Отдельный механизм для отбора резонансных новостей, которые могут "выстрелить" на массовых площадках типа Пикабу.

Целевая аудитория
Hardcore: SOC-аналитики, Blue/Red Team специалисты, системные администраторы, разработчики

Business: ИБ-директора (CISO), владельцы IT-бизнеса, которым нужно быть в курсе угроз для защиты своих инфраструктур.

Education: Студенты и начинающие специалисты, для которых канал становится ежедневным источником для самообразования.

# Project: Digital Whisper (Цифровой Шепот) - Omni-Channel CyberSec AI Aggregator
# Role: Senior Python Developer & DevOps Engineer

## 1. Project Overview
Необходимо разработать отказоустойчивое Python-приложение для автоматического сбора новостей кибербезопасности из RSS-лент, их перевода и рерайта через OpenAI API (gpt-4o-mini), модерации через Telegram-бот администратора и последующего мультипостинга на 6 платформ: Telegram, ВКонтакте, X (Twitter), LinkedIn, Threads, а также генерации еженедельных дайджестов для Хабра и фильтрации виральных новостей для Пикабу. Приложение должно быть полностью контейнеризировано с помощью Docker и оснащено продвинутой системой логирования и алертинга.

## 2. Tech Stack
* **Language:** Python 3.11+
* **Framework:** `aiogram` (v3) для асинхронного Telegram-бота модерации.
* **Database:** `sqlite3` (хранение новостей, оригинальных URL, типов текстов, статусов публикации и флагов популярности).
* **Logging:** `loguru` (ротация файлов, структурированный вывод).
* **Containerization:** `Docker`, `Docker Compose`.
* **Libraries:** `feedparser`, `openai`, `vk_api`, `tweepy`, `python-dotenv`, `apscheduler`, `aiohttp`.

## 3. Directory Structure
Приложение должно иметь модульную структуру, готовую к локальному запуску в изолированном контейнере:
```text
